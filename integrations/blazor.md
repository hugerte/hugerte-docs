This file will provide documentation for the [HugeRTE Blazor integration package](https://github.com/hugerte/hugerte-blazor).

For now, please visit the TinyMCE Blazor integration docs:

- https://www.tiny.cloud/docs/tinymce/6/blazor-cloud/
- https://www.tiny.cloud/docs/tinymce/6/blazor-ref/

Important: Write hugerte instead of tinymce and HugeRTE instead of TinyMCE/TinyMce everywhere and review our [changelog](https://github.com/hugerte/hugerte-blazor/blob/master/CHANGELOG.md) as we changed/removed some props. The HugeRTE Blazor integration nuget package name is `HugeRTE.Blazor`.

If you don't pass a ScriptSrc, HugeRTE will be loaded from jsDelivr CDN and you can specify the version using the CdnVersion prop. If you want to self-host HugeRTE, you can use the [HugeRTE nuget package](https://www.nuget.org/packages/HugeRTE) (HugeRTE v1.0.10 and later versions are published to nuget).
